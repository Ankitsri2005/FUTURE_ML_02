# 🎯 TicketIQ — Intelligent Support Triage System
> Classify. Prioritize. Respond. Faster.

## Overview
ML-powered support ticket classifier that automatically
categorizes customer complaints and assigns priority levels.

## Features
- Category classification (86.68% accuracy)
- Priority prediction (95.82% accuracy)
- Confidence score on every prediction
- Auto response suggestion
- Bulk CSV analysis dashboard
- Rule + ML hybrid priority detection

## Tech Stack
- Python, NLTK, Scikit-learn
- Flask, HTML, CSS
- TF-IDF Vectorization
- Logistic Regression (best model)

## Model Results
| Model | Category | Priority |
|---|---|---|
| Logistic Regression | 86.68% ✅ | 95.82% ✅ |
| Random Forest | 85.14% | 94.44% |
| Naive Bayes | 82.65% | 74.77% |
